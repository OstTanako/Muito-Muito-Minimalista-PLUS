**Muito, Muito Minimalista: PLUS** is a arcade spinoff project based on the original game **Muito, Muito Minimalista**.
This spinoff not only port the levels to an engine but also upgrade the art and add new mechanics and restructures the basic mechanics.



# Moding the Project

For you to modify the project you need:

1. Godot Engine 4.6+ Standard. 
2. Clone the repository with git (recommended)
``` bash
cd  /your/destined/path/here

git clone https://github.com/OstTanako/Muito-Muito-Minimalista-PLUS.git
```
**⚠️ • Be warned that this project uses a customized license, I recommend you read it before you do anything.**

Also, Standard version means **no .NET support**.

This project it's begin developed with official permission from the original creator of MMM (Erilab / Erickseen).
